# Node & npm Introduction

This project was created while learning how to work with Node.js and npm.  
I practiced initializing a Node project, managing dependencies, and running JavaScript outside the browser.

## What I learned
- How to initialize a Node project using `npm init`
- Understanding the role of `package.json`
- Installing and managing packages with npm
- Running JavaScript with Node.js
- Basic project structure for Node applications

This repository represents my first steps into backend JavaScript and the Node ecosystem.
